# Temperature Monitoring System (C)

A simple C-based simulation project that generates random temperature values and checks if any reading crosses a user-defined threshold.

## Features
- Random temperature generation
- Threshold alert
- Loop-based monitoring
- Lightweight logic suitable for embedded simulations

## Tech Stack
- C Programming
- Standard Library
- Basic Embedded Logic Simulation
